# Automação de Testes - Coffee Cart (Cypress)

Este projeto consiste na automação de testes E2E da aplicação **Coffee Cart** utilizando o framework **Cypress** com **Cucumber** e **Page Objects**.

## Pré-requisitos

Para executar o projeto, você precisa ter instalado e configurado corretamente em sua máquina:

* **Node.js**
* **Git**
* **IDE (VS Code recomendado)**

## Configuração da IDE

Recomenda-se instalar as seguintes extensões para melhor visualização e escrita dos arquivos `.feature`:

* **Cucumber (Gherkin) Full Support**

## Instalação e Configuração

Antes de rodar, instale as dependências:

1. Instale as dependências do projeto:
   * `npm install`

## Como Rodar

Para abrir a interface interativa do Cypress:

  * `npx cypress open`
